Проекты

GPT-3

Третье поколение алгоритма обработки естественного языка от OpenAI

Подробнее

Самая крупная и продвинутая языковая модель в мире, способная решать любые задачи на английском языке. По сравнению с GPT-2 количество используемых параметров увеличилось более чем в 100 раз: с 1,5 до 175 млрд. Обучение модели происходило на суперкомпьютере Microsoft Azure AI, который был построен специально для OpenAI.

DALL·E

GPT-3 с 12 миллиардами параметров, обученная генерировать изображения из текстовых описаний

Подробнее

Может создавать изображения из текстовых описаний. Обладает разнообразным набором возможностей, включая создание антропоморфизированных версий животных и объектов, правдоподобное объединение несвязанных концепций, рендеринг текста и применение преобразований к существующим изображениям.

CLIP

Универсальный классификатор изображений, способный решать практически любую задачу

Подробнее

CLIP (Contrastive Language–Image Pre-training) основан на большом объеме работы по передаче нулевых кадров, наблюдению за естественным языком и мультимодальному обучению. Она обучается на самых разнообразных изображениях с широким спектром естественного языка, который в изобилии доступен в Интернете.

CaseText
Casetext автоматизирует судебные задачи, чтобы помочь адвокатам эффективно обеспечивать высококачественное представительство, предлагая комплексную платформу для юридических исследований, используемую более чем 5500 юридическими фирмами. Платформа включает в себя CARA A. I., технологию, которая автоматизирует важнейшие задачи юридических исследований, и Compose, первую в своем роде технологию, которая автоматизирует существенные элементы судебного разбирательства. С помощью технологии OpenAI Casetext стремится расширить свои возможности семантического поиска, что может сэкономить юристам сотни часов на исследованиях.

Создание контента

X

Generation

Средства создания разностроннего контента

OpenAI API может генерировать сложный и последовательный естественный язык и позволяет использовать такие варианты использования, как литературное творчество.

AI Dungeon
AI Dungeon-это текстовое приключение на основе искусственного интеллекта, где каждый ответ определяется моделью языка искусственного интеллекта. Как правило, для этих типов игр разработчик должен предварительно запрограммировать дерево решений и текстовые параметры для выбора пользователем. AI Dungeon - это первая игра в своем роде, в которой возможен любой вариант истории, и ИИ адаптирует приключение к тому, что ввел пользователь.

Первоначально построенный на GPT-2, после перехода на новую технологию OpenAI, доступную через API, AI Dungeon значительно увеличил вовлеченность пользователей и количество подписок.

Повышение производительности

Productivity Tools

Средства повышения производительности

OpenAI API позволяет анализировать текст в таблицах электронных таблиц, обобщать обсуждения по электронной почте, расширять контент из маркированных пунктов и многое другое.

Natural Language Shell
API можно использовать для перевода естественного языка в команды unix, используя несколько репрезентативных примеров.

Spreadsheets
API может сгенерировать таблицу с предлагаемыми категориями, чтобы упростить извлечение и организацию информации.

Code Completion
С помощью API мы можем генерировать полезные и контекстно-зависимые предложения по коду. После точной настройки кода из тысяч репозиториев GitHub с открытым исходным кодом API завершает код на основе имен функций и комментариев.
